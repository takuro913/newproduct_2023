## 苦戦した点

- ※めちゃくちゃ言い訳します。見苦しいので飛ばしてください。自分のために書き残しておきます。

要約すると、スケジュールの組み方、エラー、ネットの情報を基に自分のコードを書き換えることに失敗、苦戦しました。

「課題について」：そもそもはlaratterを基に画像アップロードの機能やコメント機能を付け加えて筋トレをする人のための筋トレ日記的な物を作りたかった。筋トレの様子や身体の仕上がり具合を写真で撮って記録、共有できたら...いろんな人からコメントを受け取ってアドバイスやうれしい言葉を貰えたら...と思って作り始めたが甘かった。度重なるエラーによってなかなか先へ進めず。結局ごちゃごちゃになりすぎて、講義で作ったlaratterに1つ機能を付け加えるようにしようとなったのが10/1だった。遅すぎた。1つに絞った機能である画像アップロードの機能もネットで調べたとおりに書き、自分のコードに合わせて変えるべき所は変えたつもりだったのだがうまくいかず、結果画像アップロードの機能は実現しきれなかった。カタパルトの皆さんやチューターさんが本当に熱心にサポートして頂いたこともあって悔しい結果となった。しかしlaravelを嫌いにはならなかった。
「全体を通して」：分からないところやエラー部分はネットやYouTubeを中心に調べていたが、自分のコードに置き換えるのが大変だった。似たようなコードやそれっぽいコードなのは分かったのだが、もちろんその記事を書いている人の変数やカラム名は自分の物と違う。さらにターミナルではバージョンの違いなのかコマンドを実行できないものがあった。phpのそもそもの学習不足、ググり力を高めたい。
